# All-In-One 스포츠

## 프로그램 설명 
### 스포츠 팬들이 서로 여러 사이트를 돌아보는 것이 아니라 스포츠에 모든 것을 하나의 플랫폼에서! 
#### 1. 여러 스포츠를 하나의 플랫폼에서 연계
#### 2. Follow 기능을 통해 자신이 좋아하는 팀의 소식을 가장 먼저 받기
#### 3. 경기 일정을 캘린더로 한눈에 보기
#### 4. 티켓 양도와 동행을 플롯폼 내 커뮤니케이션 기능을 통해 소통하기
#### 5. 1대기로 대화가 가능한 쪽지 기능
#### 6. 티켓 예약도 모두 하나의 플랫폼에서 해결하기

## 프로그램 기능표

| 기능  | 설명        예시                                 |
|-----|----------------------------------------------|
| 계정  | 회원가입 및 로그인                                   |
| 팀 팔로우 | 좋아하는 팀의 소식을 우선적으로 받기         |
| SNS 연계 | 좋아하는 팀의 SNS 계정 하이퍼링크 연계 |
| 캘린더 | 전체 스포츠 경기 일정 및 내가 팔로우한 팀의 경기 일정 등을 캘린더로 확인   |
| 게시판 | 양도 및 동행을 원하는 사람들끼리 게시판을 통해 서로 소통 및 댓글 기능     |
| 메시지 | 양도 및 동행에서 1 대 1로 소통이 필요한 경우 쪽지 보내기를 통해 대화 가능 |
| 하이퍼링크 | 팀별 티켓 예매 사이트 연계                              |

# 프로그램 기능

## < 계정 >

### 1. 아이디
- [X] 아이디 정보 저장
- [X] 영문자와 숫자(정수)만 존재하는지 검증
- [X] 최대길이(미정)를 넘지 않는지 검증
- [X] 공백이 아닌지 검증

### 2. 비밀번호
- [X] 비밀번호 저장
- [X] 영문자와 숫자만 존재하는지 검증
- [X] 최대길이(미정)를 넘지 않는지 검증
- [X] 공백이 아닌지 검증

### 3. 선호종목 목록
- [X] 선호 종목들을 미리 정의해놓는다

### 4. 선호팀 목록
- [X] 선호팀들을 미리 정의해놓는다

### 5. 회원정보
- [X] 아이디와 비밀번호를 저장한다
- [X] 회원은 일반 < 관리자 중 하나를 저장합니다 `[어떻게 지정할지는 미정]`
- [X] 고유의 키값(토큰)을 저장합니다
- [X] 선호하는 종목(복수 선택 가능)을 저장합니다
- [X] 선호하는 팀(복수 선택 가능)을 저장합니다

### 6. 회원가입
- [X] 아이디와 비밀번호를 입력받아 회원정보로 데이터를 넘긴다
- [X] 미리 정의된 선호 종목/팀 중에서 선택한 데이터를 회원 정보로 넘긴다
- [X] 토큰을 생성하여 데이터를 회원정보에 넘긴다

### 7. 로그인
- [X] 아이디와 비밀번호를 입력받는다
- [X] 로그인 검증 -> 실패 시 ERROR메시지 출력 및 재입력

### 8. 입력
- [X] 아이디(이메일 또는 닉네임)를 입력받는다
- [X] 비밀번호를 입력받는다
- [X] 선호 종목 / 선호 팀을 입력받는다

### 9. 마이페이지 화면
- [X] 내 정보 보기(닉네임, 팔로우한 팀 목록, 쪽지함)
- [X] 비밀번호 변경, 닉네임 변경

#### 내 정보 보기
- [X] 닉네임, 이메일, 팔로우한 팀 목록, 쪽지함 표시

#### 비밀번호 변경
- [X] 변경 권한 검증
- [X] 비밀번호 입력 회원정보에 데이터 넘기기

< 팀 팔로우 >
### 팀 정보
- [X] 스포츠 종류 저장
- [X] 팀 이름, 로고, 리그 저장
- [X] 경기 일정 저장
- [X] SNS정보 저장

### 팀 목록 화면
- [X] 스포츠 -> 팀 목록 화면
- [X] 팀 로고, 이름, 리그 표시
- [X] 각 팀 카드/행에 [팔로우] / [언팔로우] 버튼

### 팔로우
- [X] 팔로우 버튼 이벤트 발생 시 회원 정보 업데이트 하기
- [X] 언팔로우 버튼 이벤트 발생 시 회원 정보 업데이트 하기

< SNS 연계 >
### 팀 상세 페이지
- [X] 팀 기본 정보 표시
- [X] SNS 하이퍼링크
- [X] 상단 혹은 가장자리에 팔로우한 팀 목록 표시

< 캘린더 >
### 경기 일정 캘린더
- [X] 캘린더 화면(필터링 기능 포함)
  + 팀별 / 경기 유무에 따른 표시
- [X] 팀별 일정 조회

< 게시판 >
#### 게시판 글
- [X] 제목, 작성자, 등록일, 경기 정보, 조회수, 현재 상태(거래 중/거래 완료)를 저장한다
- [X] 글 내용을 저장한다
- [X] 글에 따른 키값을 생성한다

#### 댓글
- [X] 제목, 본문, 작성자, 작성시간, 좋아요 정보 저장

#### 게시판
- [X] 화면 UI 지정
- [X] 필터링 기능 (전체, 양도, 동행)
- [X] 검색 기능(제목, 작성자 기준)
- [X] 정렬 기능(최근 작성 순 등)
- [X] 게시글들 표시
- [X] 글 작성/수정/삭제

#### 글 작성
- [X] 게시판 글 작성 UI 지정
- [X] 제목, 작성자, 등록일, 경기 정보, 조회수, 현재 상태(거래 중/거래 완료) 입력값 받아 반환
